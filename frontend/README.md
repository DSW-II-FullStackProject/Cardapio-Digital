# 🍕 Tri Fratelli Frontend

## 🍽️ Sobre o Projeto
Frontend do cardápio digital Tri Fratelli, feito em React + TypeScript + CSS, integrado ao backend.

## ⚙️ Tecnologias
React · TypeScript · Vite · CSS

### 🚀 Rodando o Projeto
```
git clone https://github.com/seuusuario/Project.git
cd frontend
npm install
npm run dev
```
